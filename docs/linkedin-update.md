## About

I am an Azure Data Engineer with more than three years of professional experience and a current focus on building reliable data pipelines and lakehouse processing solutions using Azure Data Factory, Azure Databricks, PySpark, Delta Lake, ADLS Gen2 and Microsoft Fabric.

My work and portfolio projects cover metadata-driven orchestration, Landing-to-Raw processing, schema and record validation, framework logging, watermark controls, rerun handling, Unity Catalog integration and downstream data readiness.

I approach data engineering with a production operations mindset. A pipeline should not only move data successfully; it should also provide clear validation, monitoring, failure handling and recovery behaviour.

Selected portfolio work includes:

- A quarterly ADF and Databricks platform processing 14 configured source tables into validated Raw Delta tables
- A validation-first Microsoft Fabric Lakehouse pipeline with quarantine handling and controlled file lifecycle
- A metadata-driven ADF ingestion framework supporting SQL, REST and file-based sources
- Databricks-based retail validation and ADF Mapping Data Flow case studies
